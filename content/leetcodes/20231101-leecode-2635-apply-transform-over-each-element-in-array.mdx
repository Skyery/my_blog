## 題目分析
給一整數陣列 **arr** 以及一個函式 **fn**，傳回一個新陣列，對每個元素進行轉換。  
傳回的陣列應該被新建，使得 **returnedArray[i] = fn(arr[i], i)**。  
不能使用內建的 **Array.map** 方法。

## 最終結果

```js:apply-transform-over-each-element-in-array.js
/**
 * @param {number[]} arr
 * @param {Function} fn
 * @return {number[]}
 */
var map = function(arr, fn) {
    const result = []

    for(const [index, a] of arr.entries()){
        result[index] = fn(a, index)
    }
    return result
};
```
