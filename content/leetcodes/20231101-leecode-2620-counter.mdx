## 題目分析
給一整數 **n**，傳回一個計數器函式。  
計數器最初傳回 **n**，之後每次呼叫時都傳回比先前的值多 **1**  
(n、n + 1、n + 2 ..)。

## 最終結果

```js:counter.js
/**
 * @param {number} n
 * @return {Function} counter
 */
var createCounter = function(n) {
    return () => n++;
};
```
