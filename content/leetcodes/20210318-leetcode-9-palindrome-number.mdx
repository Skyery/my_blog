## 題目分析
給一正整數 **x**，若 **x** 是 **palindrome** 則回傳 **true**，否則回傳 **false**。

## 最終結果

```js:palindrome-number.js
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    var arr = x.toString().split('');
    var a = arr.slice(0).reverse();
    if (JSON.stringify(arr) === JSON.stringify(a)) {
        return true;
    } else {
        return false;
    }
};
```
