## LeetCode 練習 - 1. Two Sum
> Javascript  
> 日常刷題練習

## 題目分析
給定一個整數陣列 **nums** 和一個整數 **target**，  回傳兩個陣列索引，使其將加等於 **target**。  
不能重複同一個索引。  
可以假設每個輸入都有一個解。  
可以任意排序回傳結果。

## 最終結果

```js:two-sum.js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    const hashTable = new Map();

    for(let i = 0; i < nums.length; i++){
        const diff = target - nums[i];

        if(hashTable.has(diff)) return [i, hashTable.get(diff)];

        hashTable.set(nums[i], i);
    }

    return [];
};
```
