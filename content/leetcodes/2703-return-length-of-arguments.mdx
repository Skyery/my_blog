## LeetCode 練習 - 2703. Return Length of Arguments
> Javascript

## 題目分析
寫一個名為 argumentsLength 的函式，傳回傳遞給函式的參數數量。

## 最終結果

```js:return-length-of-arguments.js
/**
 * @param {...(null|boolean|number|string|Array|Object)} args
 * @return {number}
 */
var argumentsLength = function(...args) {
    return arguments.length;
};
```
