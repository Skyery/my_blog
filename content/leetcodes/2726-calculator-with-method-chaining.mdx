## LeetCode 練習 - 2726. Calculator Method Chaining
> Javascript  
> Javascript 30 天系列

## 題目分析

設計一個 **Calculator Class**。該類應提供加法、減法、乘法、除法和指數運算的數學操作。它還應允許使用方法鏈進行連續操作。**Calculator Class** 類的構造函數應接受一個數字，該數字作為結果的初始值。

你的 **Calculator Class** 應該具有以下方法：

* **add** - 此方法將給定的數值加到結果上，並傳回更新後的 **Calculator**。
* **subtract** - 此方法將給定的數值從結果中減去，並傳回更新後的 **Calculator**。
* **multiply** - 此方法將結果乘以給定的數值，並傳回更新後的 **Calculator**。
* **divide** - 此方法將結果除以給定的數值，並傳回更新後的 **Calculator**。如果傳入的值為 **0**，應拋出錯誤 **"Division by zero is not allowed"**。
* **power** - 此方法將結果提高到給定數值的冪，並傳回更新後的 **Calculator**。
* **getResult** - 此方法傳回結果。  

在 **10^-5** 範圍內的解決方案被認為是正確的。

## 最終結果

```js:calculator-with-method-chaining.js
class Calculator {
    
    /** 
     * @param {number} value
     */
    constructor(value) {
        this.num = value;
    }
    
    /** 
     * @param {number} value
     * @return {Calculator}
     */
    add(value){
        this.num += value;
        return this;
    }
    
    /** 
     * @param {number} value
     * @return {Calculator}
     */
    subtract(value){
        this.num -= value;
        return this;
    }
    
    /** 
     * @param {number} value
     * @return {Calculator}
     */  
    multiply(value) {
        this.num *= value;
        return this;
    }
    
    /** 
     * @param {number} value
     * @return {Calculator}
     */
    divide(value) {
        if(value === 0) throw new Error("Division by zero is not allowed");
        this.num /= value;
        return this;
    }
    
    /** 
     * @param {number} value
     * @return {Calculator}
     */
    power(value) {
        this.num **= value;
        return this;
    }
    
    /** 
     * @return {number}
     */
    getResult() {
        return this.num;
    }
}
```
