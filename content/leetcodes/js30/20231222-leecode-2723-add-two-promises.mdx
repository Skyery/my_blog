## LeetCode 練習 - 2723. Add Two Promises
> Javascript  
> Javascript 30 天系列

## 題目分析
給兩個 **Promise** 分別為 **promise1** 和 **promise2**，傳回一個新的 **Promise**。  
**promise1** 和 **promise2** 都將 **resolve** 一個數字。  
傳回的 **Promise** 應該 **resolve** 兩個數字的和。

## 最終結果

```js:add-two-promises.js
/**
 * @param {Promise} promise1
 * @param {Promise} promise2
 * @return {Promise}
 */
var addTwoPromises = async function(promise1, promise2) {
    const [p1, p2] = await Promise.all([promise1, promise2]);
    return p1 + p2;
};
```
