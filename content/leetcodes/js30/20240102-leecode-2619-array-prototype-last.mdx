## LeetCode 練習 - 2619. Array Prototype Last
> Javascript  
> Javascript 30 天系列

## 題目分析

寫一段程式碼，可以擴增所有陣列，使你可以在任何陣列上調用 **array.last()** 方法，並傳回最後一個元素。如果陣列中沒有元素，則應返回 -1。

你可以假設該陣列是 **JSON.parse** 的輸出。

## 最終結果

```js:array-prototype.js
/**
 * @return {null|boolean|number|string|Array|Object}
 */
Array.prototype.last = function() {
    return this.length === 0 ? -1 : this.at(-1);
};

/**
 * const arr = [1, 2, 3];
 * arr.last(); // 3
 */
```
