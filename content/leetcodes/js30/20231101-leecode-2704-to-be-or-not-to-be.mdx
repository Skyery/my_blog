## 題目分析
寫一個函式 **expect**，接受字串 **val** 並傳回一個具有以下兩個函式的物件。  

* **toBe** 接受另一個值，如果兩值相等 **===**，則傳回 **true**。如果不相等，則拋出錯誤 **"Not Equal"**。  
* **notToBe** 接受另一個值，如果兩值不相等 **!==**，則傳回 **true**。如果他們相等，則拋出錯誤 **"Equal"**


## 最終結果

```js:to-be-or-not-to-be.js
/**
 * @param {string} val
 * @return {Object}
 */
var expect = function(val) {
    return {
        toBe: (compare) => {
            if(val !== compare) throw new Error("Not Equal");
            return true;
        },
        notToBe: (compare) => {
            if(val === compare) throw new Error("Equal");
            return true;
        } 
    }
};
```
