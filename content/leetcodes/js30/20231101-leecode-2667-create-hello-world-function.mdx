## 題目分析
寫一函式 **createHelloWorld**。  
回傳新函式且始終回傳 **"Hello World"**。

## 最終結果

```js:create-hello-world-function.js
/**
 * @return {Function}
 */
var createHelloWorld = function() {
    return function(...args) {
        return "Hello World"
    }
};
```
