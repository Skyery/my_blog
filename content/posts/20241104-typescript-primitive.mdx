## TS 基本型別 (Primitive Types)

### number

`number` 型別代表所有數字類型的數據，包括整數、浮點數、正負無窮大、NaN（非數值）等。

```ts

const age: number = 34;             // 整數
const temperature: number = 36.3;   // 浮點數
const notANumber: number = NaN;     // NaN
const infinity: number = Infinity;  // 無限大
```

### string

`string` 型別表示一個由字符組成的字串。

```ts
const name: string = "Jason Lin";
```

### boolean

`boolean` 型別只有兩個可能的值：`true` 或 `false`，用來表示二選一的狀態。

```ts
const hasPets: boolean = true;
```

### symbol

`symbol` 是一個新的原始型別，代表唯一的標識符。每個 `symbol` 值都是唯一的，通常用於對象屬性名的唯一標識。

```ts
const uniqueID: symbol = Symbol("id");
```

### bigint

`bigint` 型別用來表示任意精度的整數，適合需要處理超過 `Number` 類型的範圍的情況。

```ts
const largeNumber: bigint = 9007199254740991n;
```
