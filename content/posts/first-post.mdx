# 歡迎來到我的新部落格！

這個部落格使用了 Next.js App Router、Tailwind CSS 和 Contentlayer。

## 功能亮點

- Markdown/MDX 內容處理
- 自動 Slug 生成
- 漂亮的代碼高亮 (由 `rehype-pretty-code` 提供)

```tsx
import React from 'react';

function HelloWorld() {
    return <h1>Hello, World!</h1>;
}

export default HelloWorld;
```

希望你喜歡這裡！
